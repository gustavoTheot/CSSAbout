Cores em css

As palavras-chave de cor são identificadores não sensíveis a maiúsculas e minúsculas que representam uma cor específica, ex.: red, blue, brown, ou lightseagreen.

CSS nível 1 aceita apenas16 cores básicas, denominadas cores VGA já que foram tiradas do conjunto de cores exibíveis das placas gráficas VGA.
CSS nível 2 Adicionou a palavra-chave orange.

Unidades

px

É a unidade mais utilizada nas paginas web.

rem

É considerada uma medida relativa. O rem é o sucessor do em mas sua implementação é diferente

em

Unidade relativa ao tamanho da fonte do elemento pai.

vh & vw

O vh (viewport height) representa o tamanho da altura da tela visível e o vw (viewport width).

calc

É uma função do CSS que retorna um valor com base no cálculo efetuado. Pode combinar unidades.
